# CN-001: Tool-Scoped Progressive Disclosure

> **This concept has been absorbed into [PHILOSOPHY.md](../PHILOSOPHY.md).**

*Tool-Scoped Progressive Disclosure* is the formal academic name for the methodology now called **Distributed Prompting** (implementation pattern: **Prompt-On-Call**). In public-facing surfaces we use "Distributed Prompting" and "Prompt-On-Call"; this file retains the original name for citation continuity.

CN-001 was the formal theoretical anchor for Talking CLI's methodology. Its content — the four-layer model (L1–L4), boundary rules, anti-patterns, canonical example, and adoption guidance — is now fully documented in PHILOSOPHY.md as the Four Channels (C1–C4) and Four Rules of Talk.

Layer mapping preserved for citation: L1 = C1 (Contract), L2 = C2 (Handshake), L3 = C3 (Voice), L4 = C4 (Judgment).
